# Language-translator-using-python-with-GUI
This project is a Language Translator application built with Python and a Graphical User Interface (GUI). It allows users to easily translate text between multiple languages in a simple, interactive environment.
It's features are :
User-friendly GUI: Built using Tkinter (or your chosen GUI library).

Multi-language support: Translate text across a wide range of languages.

Real-time translation: Instant results with just a click.

Copy & clear options: Manage translated text efficiently.

Lightweight & portable: Easy to run on any system with Python installed.

Technology used :
Python – Core programming language

Tkinter – GUI framework

Googletrans – For language translation
